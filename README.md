# עצומת תושבי רחוב בר יהודה, קדימה

דף נחיתה לאיסוף חתימות תושבים, בדרישה למועצה האזורית קדימה-צורן לטיפול מיידי
במפגעי הבטיחות התחבורתיים ברחוב בר יהודה — לאחר תאונה קשה וכחמש שנות התרעות.

**הדרישות:** הפיכת הרחוב לחד-סטרי · פסי האטה · מראה פנורמית בעיקול · הסדרת חנייה.

האתר חי בכתובת: https://tomhagiladi.github.io/bar-yehuda-kadima/

מנגנון החתימות נשמר לגיליון Google Sheets פרטי דרך Google Apps Script (ראו `backend/Code.gs`).
